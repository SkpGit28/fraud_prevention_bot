## Bot Features Explained

### **Feature 1: Recovery Guide** (Most Important)
**When:** User has been scammed
**Example:** "I lost ‚Çπ50,000 via UPI fraud"
**Bot provides:**
- Exact phone number to call (bank)
- What to say (word-by-word)
- When to call (immediately)
- What happens next (timeline)
- Success probability (60-80%)
- Then what (1930 hotline, cybercrime.gov)
**Result:** Many victims recover their money! üí∞

### **Feature 2: Number Checker**
**When:** Senior gets suspicious call/message
**Example:** "Check +91-9876543210"
**Bot responds:**
- Is it a known scam?
- What type of scam?
- How many people reported it?
- What are the red flags?
- What to do?
**Result:** Fraud prevented before it happens! üõë

### **Feature 3: Scam Reporter**
**When:** Senior wants to warn others
**Example:** "Report romance scam, someone pretending to be my son"
**Bot:**
- Records the report
- Encrypts all data
- Waits for community verification
- After 50+ confirmations ‚Üí Warns all seniors
**Result:** Seniors protect each other! ü§ù

### **Feature 4: Help Menu**
**When:** User types "help"
**Bot shows:**
- 4 main commands
- How to use each
- Government helpline (1930)
- Official website (cybercrime.gov)
**Result:** Easy navigation for seniors! üì±
